## API
### Player Movement
* `player.moveForward()`
* `player.moveBackward()`
* `player.moveLeft()`
* `player.moveRight()`
* `player.stopMoving()`

* `player.turnLeft()` _Alias: `player.rotateCounterClockwise()`_
* `player.turnRight()` _Alias: `player.rotateClockwise()`_
* `player.stopTurning()` _Alias: `player.stopRotating()`_

* `player.stop()` - Stop moving and stop turning
